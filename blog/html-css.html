What does it mean to display inline vs inline blocks?

Inline and Inline Blocks are the display property of how an element can be displayed in a webpage. These are the two frequently used building blocks of web pages. In this post we will discuss and recognise how these elements behave in building the layouts.

Inline

Inline elements are displayed in same line, just as the name suggests. This means there is no line breaks and they can line up next to other elements on the same line. Therefore, it takes up as much width as necessary.They are not affected by the change in height or width properties. But, they can still have an effect with the margins and with padding. Margins can only only work horizontally left or right, but not vertically. Padding can be applied on all sides, but the top and bottom may overlap other elements.

Inline Blocks

Inline block is essentially same as Inline except you have the ability to adjust the height and width properties. Also, it does not add a line-break so its on the same l ine as other elements. Padding and Margins can be added on all four sides. CSS code to display is: inline-block.
